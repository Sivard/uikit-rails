# UIKit Sass for Rails

UIKit Sass for Rails Implementation

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'uikit-rails'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install uikit-rails
